This program calculates and produces plots of the band structures of armchair (AGNR) and zigzag (ZGNR) 
graphene nanoribbons (GNRs) using the tight-binding TB model. Users can select 1st, 2nd or 3rd 
nearest-neighbour hopping interactions as well as the width N of the GNR system.

Usage: 
    Execute the code to be prompted for user input.

Dependencies:
    numpy
    matplotlib
